# 🍎 Apple Falling Game

## 📌 Description
Apple Falling Game is a fun 2D Unity mini game inspired by the famous Newton apple story and the struggles students face while studying Physics 😄.

In this game, the player controls a character representing Isaac Newton and must move left and right to avoid randomly falling apples. If an apple hits Newton’s head, the game ends.

This project was created as a beginner-friendly Unity game to practice 2D game development concepts including player movement, collision detection, sound effects, UI design, and game mechanics.

---

## 🎮 Gameplay
- Control Newton using keyboard keys
- Avoid randomly falling apples
- Survive as long as possible
- Game Over when an apple hits Newton’s head

---

## ✨ Features
- 2D player movement
- Random apple spawning system
- Collision detection
- Game Over screen
- Sound effects and background music
- Simple UI design

---

- Unity Engine

- Unity 2D Tools

## 📂 Project Structure
- **Scenes** → Main menu and gameplay scenes
- **Scripts** → Player movement and game logic
- **Materials/Sprites** → Characters, apples, backgrounds
- **Sound** → Background music and effects

---

## ▶️ How to Play
1. Open the project in Unity
2. Load the main game scene
3. Press Play
4. Use Left and Right Arrow Keys to move Newton
5. Avoid falling apples and survive
